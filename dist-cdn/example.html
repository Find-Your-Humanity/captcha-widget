3. API 키 발급

위젯을 사용하려면 먼저 API 키를 발급받아야 합니다:

  1. 대시보드에 로그인
  2. API 키 생성
  3. 도메인 설정 (선택사항)
  4. 발급받은 Site Key를 위 코드의 'YOUR_SITE_KEY' 부분에 입력

4. 서버 측 검증

클라이언트에서 캡차를 성공적으로 완료한 후, 서버에서 토큰을 검증해야 합니다:

POST https://api.realcatcha.com/api/verify-captcha Content-Type: application/json { "site_key": "YOUR_SITE_KEY", "secret_key": "YOUR_SECRET_KEY", "captcha_token": "TOKEN_FROM_WIDGET", "response": "CAPTCHA_RESPONSE" }
